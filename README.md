# leet-code

Leet code problems solutions.

## Development

Installation:

```bash
npm install
```

Run unit tests:

```bash
npm test
```

Run benchmarks:

```bash
npm run bench
```

To filter tests or benchmarks use `-t` option. For example:

```bash
npm run bench -- -t rotate array
```

You can quickly generate basic structure for any problem using:

```bash
npm run cli
```
